Backup Values

You don't have to jump straight to a default value like cursive or sans-serif: you can tell CSS to try several, going from one to the next if the one you want isn't available.

For example, if you write about fonts. CSS will first try to apply Verdana to your paragraphs. If the user's computer doesn't have that font, it will try Tahoma next, and if that doesn't work, it will show a default sans-serif font.

< font-family: Verdana, Tahoma/> (first write the original value and then the backup value)

Universal Selector

There's also a very special selector you can use to apply CSS styling to every element on the page: the * selector. For example, if you type

    * { 
    border: 2px solid black; 
    }
You'll create a two-pixel wide solid black border around every element on the HTML page.

Taking Up Space

Each HTML box gets its own space using 4 methods:

  1. block: This makes the element a block box. It won't let anything sit next to it on the page! It takes up the full width.
  2. inline-block: This makes the element a block box, but will allow other elements to sit next to it on the same line.
  3. inline: This makes the element sit on the same line as another element, but without formatting it like a block. It only takes up as much width as it needs (not the whole line).
  4. none: This makes the element and its content disappear from the page entirely!

But how to use it?

In the CSS tab, just like block, one can use other spacing properties too (like: none and inline).

Margin: Space around the element.

There are 2 ways to set the margin

  • margin-top: (value) margin-right: (value) margin-bottom: (value) margin-left: (value) or (value, value, value, value)
  • .margin: auto

Border: Edge of the element. It's what we've been making visible every time we set the border property. They can be set just like margin. Either by putting all the values together (value, value, value, value) or one by one.

Padding- spacing between the content and the border. They can be set just like margin. Either by putting all the values together (value, value, value, value) or one by one.

Moving the values to the negative side : moving an element in the other direction, one can give CSS a negative value:

For instance, margin-left: -20px will move the element twenty pixels to the left.

Floating and Clearing

FLOATING: It helps to position the elements such that the other elements floating do not get disturbed and all of them don’t land on top of each other.

The div used is float. It can be written as for - Right or Left.

CLEARING: If sometimes the elements come on top of each other. This is used.

clear: left : it will immediately move below any floating elements on the left side of the page; it can also clear elements on the right.

clear: both : it will get out of the way of elements floating on the left and right

NOTE : If things are not positioned it just plunks itself down in the document.

TYPES OF POSITIONING

There are 3 types of positioning-

1. Absolute : Absolute positioning is in relation to the first parent element it has that doesn't have position: static.

2. Relative : Relative positioning tells the element to move relative to where it would have landed if it just had the default static positioning.

3. Fixed : Fixed element stays put even as other elements scroll past.

The above example shows how to put the different types of positioning in the CSS tab. Also, the same method will be applied to Relative and Fixed positioning
