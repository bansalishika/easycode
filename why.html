WHY CODING?

Programming is to write a command code that tells the computer what to do when a task is given. Writing codes are usually the process of creating a program we all use today. It is crucial to learn how to code because in today’s world we are entirely dependent upon technology and it is important for a person to know how humans give commands to computers so that they work.

If a person know’s how to code, they can make any application or any website for their personal or social use. Further, it can help for the development of different fields. If you are a student and need some help in a particular subject, you know how to programme; you can create an application so as to get some support in it.

Through this website, I am covering the basics of Markup language i.e. HTML along with the cascading style sheet concepts. Also, I am focusing on the basics of a programming language jQuery. I chose these languages because I believe that before doing the hardcore programming languages, it is beneficial first to learn the markup languages.

Why learn HTML and CSS?

Everything you see on a website is a result of the combination of HTML and CSS. With these two languages, you will have the skills you need to bring your website design to life.

Why learn jQuery?

Learning jQuery is the fastest and easiest way to add interactivity and animation to your website. Not only is it a great way to get introduced to the what JavaScript can help you accomplish, but this lightweight library will also let you leverage the HTML and CSS skills you've already learned. After learning it, you will start your journey of becoming one potent front-end developer!
