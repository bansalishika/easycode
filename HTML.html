Basic Structure Of An HTML Code

To start a tag use <> tags and to end it always put / before the closing >

Indentation is always needed to write a basic HTML code.(whatever written in <> is called as tags)

     <!DOCTYPE html>     (This tells the browser what language it's reading) 
     <html>         (this starts the HTML document.) 

     <head>    
     <title> </title>     (defines the title of the document in the browser tool bar.) 

     </head>  
     <body> </body>    (This element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.) 
 
     </html>     (this closes the HTML document)

Basic Styling

<h1> - <h6> tags

<h1> is used to put headings on a website. Here, h1 tag is used for the biggest size and h6 tags are for the smallest. Using these tags is just like one uses p tags. Eg: <h1> WRITE THE HEADING </h1>

The first text is written using h1 tag, whereas the second text is written using h6 tag.

Basic styling for a paragraph

< p style=“font-size:( write the font size required in numerical form)”>

  • If one needs to change the style of a heading, use h1 tags instead of p tags.
  • Same format will be used for the following styling features:
    1. Text-align
    2. Font color
    3. Font size

To write something in Bold, use <strong> </strong> tags and for italic use <em> </em> tags. Just place the phrase in between the tags and it will be styled.

Example: If lets say, you want code to be in bold letters, you will write : I like to <strong> code </strong>

Example 2: If lets say, you want code to be in italic, you will write: I like to <em> code </em>

Format For thead Tag

The <thead> tag is used to group header content in an HTML table. The element is used in conjunction with the <tbody> and <tfoot> elements to specify each part of a table (header, body, footer) . Browsers can use these elements to enable scrolling of the table body independently of the header and footer.

    <thead>
    <tr>   (for forming a row)  
    <th>   (head name)  </th > 



    <th>  (can be added for another column)  </th>





    </tr>
    </thead>

tbody Tags

The <tbody> tag is used to group the body content in an HTML table. format for <tbody> tags is exactly like < thead>
